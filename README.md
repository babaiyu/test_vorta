# Test Vorta Siber Indonesia - Bayu Permana Putra
This project is created for my application to Vorta Siber Indonesia

## Technologies
#### Frontend
- NextJS
- Chakra UI
- React Hook Form

#### Backend
- NestJS
- MySQL ver 5

## How to run locally
#### Backend
```
$ yarn install
$ yarn start:dev
```
#### Frontend
```
$ yarn install
$ yarn dev
```
